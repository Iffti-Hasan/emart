*eMart is a basic HTML, CSS and PHP-based e-commerce web system built for the Web Technology course at

  ***<h3>ðŸŸ¦ Client (Customer) Features:</h3>***

- **User Registration & Login:** Secure authentication system that allows customers to create accounts and log in using valid credentials.
- **Product Browsing:** View all available products displayed in a clean, responsive grid or card layout for easy navigation.
- **Search Products:** Quickly find specific products using a simple text-based search function.
- **Cart Management:** Add products to the cart, modify quantities and proceed to checkout.
- **Wishlist:** Save preferred products for future purchases.
- **Orders:** Access a detailed order history, including order date, status and total amount.
- **Track Orders:** Monitor the status of placed orders (Pending, Shipped, Delivered).
- **Feedback:** Submit reviews and feedback for purchased products or general shopping experience.
- **Logout:** Securely end the customer session.

  ***<h3>ðŸŸ¦ Seller (Admin) Features:</h3>***
  
- **Login:** Secure login access for sellers.
- **Dashboard:** Displays essential information such as manage products, total orders, sales report, feedback, stock alerts and discounts.
- **Product Management:** Add, edit and delete product details including name, description, price, image and stock.
- **Order Management:** View and manage the status of customer orders.
- **Sales Report:** Access summarized sales and order data.
- **Feedback:** Review and reply feedback submitted by customers.
- **Stock Alerts:** Monitor product stock levels and receive alerts.
- **Discounts:**  Manage product discounts (redirects to Product Management page)
- **Logout:** Securely log out from the admin dashboard.
